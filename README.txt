Bhargav Portfolio (Static Website)

Files:
- index.html
- styles.css
- script.js

How to run:
1) Open index.html in any browser
or
2) Use a simple local server:
   - VS Code Live Server extension, or
   - python -m http.server 8000
   Then open http://localhost:8000

Customize:
- Replace LinkedIn/GitHub/Resume links in index.html
- Replace contact email/phone placeholders
